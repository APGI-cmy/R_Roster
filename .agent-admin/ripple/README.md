## Structure

Each log file is named `layerdown-<YYYYMMDDTHHMMSSZ>.json` and contains:

```json
{
  "timestamp": "<iso-8601>",
  "issue_number": "<number>",
  "issue_title": "<title>",
  "trigger": "issue_opened | issue_labeled | workflow_dispatch",
  "agent_files_changed": true | false,
  "action_taken": "DRAFT_ESCALATION | AUTO_MERGE_PR",
  "pr_number": "<number or null>",
  "escalation_file": "<path or null>"
}
```
